Caracteristicas de Grid en Tailwind

Grid Template Columns

Define el numero y tamaño de columnas con grid-cols-{n} o grid-cols-[pattern]

Grid Columns Span

Controla cuantas columnas ocupa un elemento con col-span-{n}

Grid Template Rows

Define fila con grid-rows-{n} o grid-rows-[pattern]

Gap

Añade espacio entre elementos con gap-{size}
